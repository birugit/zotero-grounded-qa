# Lattice for Zotero

>
> - 📖 **[README（英文）](../README.md)**
> - 🚀 **[入门指南（英文）](GETTING_STARTED.md)**

Lattice 帮助你把标注和想法从“单篇论文”的孤岛中解放出来：在一个窗口里查看整个文库的所有高亮，
按颜色／标签／关键词筛选并连同引用一起导出，再把重要的标注提升为可搜索、可打标签、可互相链接的
“想法层”（Citavi 模式），同时始终与来源保持关联。此外还包含基于 PDF 的 AI 问答，并提供可点击的
页码引用。
